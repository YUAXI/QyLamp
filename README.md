# QyLamp

> 你好我叫秋雨，请用秋雨唤醒我

## 功能

- [x] 语音识别
- [x] 接入小爱同学
- [ ] WIFI控制
- [ ] MQTT远程控制
- [x] 多种颜色任意切换(红，橙，黄，绿，青，蓝，紫，白，粉)
- [x] 亮度控制
- [x] 音量控制
- [ ] 呼吸模式
- [x] 跑马灯模式
- [x] 爆闪模式(白灯持续闪烁)
- [x] 救援模式(红灯持续闪烁发出SOS救援信号)

## 注意
